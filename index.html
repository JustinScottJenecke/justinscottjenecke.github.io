About Me

A bit more about about me and what I
typically get up to these days:

What Im doing:

These days I spend my time working as a Code Coach for CodeSpace Academy, and Ed-Tech startup aimed at equipping students with programming and web development skills used in the tech industry today.

More About Me:

My background in software is in backend development, since I started my coding journey developing desktop applications and REST APIs in Java, whilst at University at CPUT. Since I've started my current position as a Code Coach, I've focused a lot more of my time into learning frontend technologies such as React.js and Vue.js, and I have spent a lot of time understanding the foundations of JavaScript, due to the many intricacies and nuances of the language, when compared to something more traditional like Java or C#.

Future Plans:

My current goal is to explore more technologies relating to software architecture and learning more about distributed systems. I have also been getting back into the .Net framework, and learning more about databases, optimizing data queries, etc. since I find that solving the complexity typically found in that sort of environment more engaging and interesting than those typically found in UI or customer facing application development.
